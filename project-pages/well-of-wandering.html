The next controls relate to the UI elements under the canvas and will be explained from the top left to the bottom right in the way they appear.

Ball Selector

Select the ball to manipulate

Lower Numbers = Lower Frequency

Higher Numbers = Higher Frequency

Speed

Changes the speed at which the ball plays sound

Waveform Selector

Selects the type of wave the ball has

All Waveforms Selector

Changes all ball's waveforms to the already selected waveform in the previous selector

Number of Balls Changer

Changes the number of balls present

All Values Randomizer

Randomizes all values, except the number of balls

Spatial Audio Toggle

Toggles between normal audio and spatial audio

Speach Ball Recorder

Starts the recording for the speech ball
