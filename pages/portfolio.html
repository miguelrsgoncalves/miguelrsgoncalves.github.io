Single Blade Grass System (work in progress)

Being fascinated with the grass system in games like Ghost of Tsushima, I started to learn how to make my own single blade grass systems based on Unreal's Niagara particle system, trying to remake the magic of a sea of grass. This is an improvement over an older project of mine. The system currently supports more than 16 Million grass blades, which I still intend to improve and augment the visuals.

Godot

Full Dynamic Painted World (work in progress)

A work in progress of a shader system that allows to fully paint all meshes, instantiated or not, in a full game scene. The objective is to have a fully transformative game, where you can paint some parts and leave others unpainted. The system supports a transformation between a 2D stylized look and a fully rendered 3D that blends seamlessly. The system doesn't use any Render Targets, saving up on memory and enabling it to scale indefinitely.
